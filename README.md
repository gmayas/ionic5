# FrontendAngularGps

Este proyecto fue creado con Ionic: 5.4.13, Node: 13.8.0

Al momento de descargar este proyecto, instale las dependencia con npm install, para ejecutar npm run dev, la apliacion corre en: http://localhost:8100/
